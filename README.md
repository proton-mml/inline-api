## inline-backend
This is the backend server for the inline system.

## Preparing

### Install dependencies
```
  $ npm install
  $ sudo npm install nodemon -g
  ```

### Set environment variables
```
  export PGHOST=?                                        
  export PGPORT=?                                                   
  export PGUSER=?                                               
  export PGPASSWORD=?                                             
  export PGDATABASE=?                                               
  export MONGO_URL=?
```

## Running
```
  $ npm run dev
```
